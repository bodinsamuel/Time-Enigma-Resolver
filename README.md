Time-Enigma-Resolver
====================

This a simple but efficient time enigma resolver, in c#


How to ?
=======

Launch the executable on the root dir and enter your sequence.
Like this, for example:
<pre>
2, 3, 1, 4, 2, 1, 4, 2, 4, 2, 2
</pre>

Press enter.
